# 🌗 React Context API Demo

This project demonstrates the use of Context API in React for global state management.

## 🚀 Features

* Light/Dark theme toggle
* Global state management using Context API
* Clean and reusable component structure

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* Context API

## 📂 Project Structure

```
src/
 ├── context/
 │    └── ThemeContext.js
 ├── components/
 │    ├── Navbar.js
 │    └── Home.js
 ├── App.js
 └── index.js
```

## ⚙️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/context-api-demo.git
```

2. Go to project folder

```
cd context-api-demo
```

3. Install dependencies

```
npm install
```

4. Run the app

```
npm start
```

## 🧠 Concepts Used

* createContext()
* Context Provider
* useContext() Hook
